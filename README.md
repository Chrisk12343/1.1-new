# WinFormsApp1 - Login Application

## Overview
A Windows Forms application built with C# for user login and input validation.

## Project Structure
- **Login.cs** - Main login form with event handlers for user interactions
- **Login.Designer.cs** - Auto-generated designer code for form layout
- **Login.resx** - Resource file for form components

## Features
- User input validation for numeric values
- Login form interface
- Input validation and error handling

## Requirements
- .NET Framework (Windows Forms)
- Visual Studio or compatible C# IDE

## Usage
1. Open the project in Visual Studio
2. Build the solution
3. Run the application
4. Interact with the login form

## Notes
- Currently in development with validation logic being refined
- Future improvements planned for authentication functionality
